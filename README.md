# YOLO Visualizer

YOLO 模型结构可视化与分析工具，支持 **网络结构图** 和 **训练结果分析** 两大功能，双击即用，无需安装 Python 环境。

---

## 功能概览

### 1. 网络结构图

- **YAML 模型解析** — 支持解析 YOLOv6/v8/v10、RT-DETR 等主流 YOLO 架构的 YAML 配置文件
- **交互式网络拓扑图** — 自动将模型分解为 Backbone / Head / Detect 三列布局，贝塞尔曲线展示层间连接关系
- **缩放与平移** — 滚轮缩放、拖拽平移，Fit 按钮一键适应屏幕
- **节点高亮** — 点击任意节点高亮其所有前驱输入节点
- **悬浮信息卡** — 鼠标悬停显示模块名称、层索引、重复次数、参数 args 等详细信息
- **模块颜色自定义** — 点击图例色块可打开颜色编辑器，自由调整每个模块类型的显示颜色

### 2. 训练结果分析

- **多文件对比** — 支持同时打开多个 CSV 训练日志文件，多曲线同图对比
- **指标筛选** — 一键显示/隐藏任意指标列，Loss 类指标自动高亮
- **最佳值标注** — 每个指标的最优值标注在图表下方，格式为：`文件名: best=0.1234 @epoch 50`
- **悬浮数据点** — 鼠标移动显示对应 Epoch 的所有指标数值
- **曲线显隐控制** — 点击图例项可单独隐藏/显示某条曲线

### 3. 界面与体验

- **暗色/亮色主题** — 一键切换，主题偏好自动保存至 localStorage，两个页面同步生效
- **零依赖运行** — Windows x64 可执行文件，双击启动，自动打开浏览器
- **内置示例模型** — 内置 yolo26.yaml、rtdetr-x.yaml，开箱即用，无需上传文件即可体验

---

## 快速开始

1. 下载 `dist\YOLO_Visualizer\` 文件夹
2. 双击 `start.bat`
3. 浏览器自动打开 `http://localhost:5000`

> 关闭终端窗口即可停止服务。
